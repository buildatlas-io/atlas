# Atlas

**World-class open developer infrastructure for builders who want more capability, ownership, and freedom.**

Atlas starts as an open-source coding agent and grows toward a complete ecosystem for AI-native software creation: agent runtime, tool system, memory, context engine, models, workflows, and developer infrastructure.

## Mission

**Expand what's possible. Earn the world's attention.**

## Vision

Build world-class open developer infrastructure that gives builders more capability, ownership, and freedom.

## Why Atlas Exists

Modern AI coding tools are powerful, but many builders are becoming dependent on closed platforms, rising prices, limited control, and black-box workflows.

Atlas exists to explore a different path:

- Open by default
- Developer-owned workflows
- Transparent agent infrastructure
- Long-term memory and context
- World-class engineering craftsmanship

The goal is not to build another AI wrapper.

The goal is to build open infrastructure that developers can understand, extend, trust, and eventually own.

## Phase One: Atlas Agent

Phase One focuses on building the first practical version of Atlas Agent for daily personal use.

The first milestone is simple:

> Atlas can take a small coding task, understand the project, create a plan, edit files safely, run checks, and summarize what changed.

### Phase One Scope

Atlas Agent should be able to:

- Read project files
- Understand project structure
- Create task plans
- Edit files safely
- Run terminal commands
- Track changes
- Maintain project memory
- Generate implementation notes
- Help improve real projects

### Not Included in Phase One

Phase One does not include:

- Training a custom AI model
- Building a full IDE
- Building a cloud platform
- Supporting teams
- Marketplace features
- Complex plugin ecosystem
